Este proyecto está pensado como una parte de un desarrollo más grande: una página web para consultorios médicos.
El objetivo principal de este proyecto es la carga de nuevos medicos en la pagina, facilitando a la persona encargada de administrar la página, como por ejemplo un recepcionista, una forma sencilla y amigable de gestionar los medicos.

El sistema incluye un carrusel donde los clientes de la pagina de los consultorios pueden visualizar los médicos disponibles en el consultorio, junto con sus especialidades y horarios de atención.

Para facilitar la carga de nuevos médicos se desarrolló un formulario. 
Este formulario valida que todos los campos estén completos: en caso de que alguno quede vacío, aparece una alerta y el médico no se carga hasta que la información esté completa. 
Una vez agregado correctamente, el nuevo médico se muestra automáticamente en el carrusel y salta una alerta de que se cargo un medico nuevo.

Además, se incorporó un botón de modo oscuro como funcionalidad adicional. 
La preferencia seleccionada queda guardada en el Local Storage, de modo que al volver a abrir la página se conserva el último modo utilizado.
